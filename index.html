TODO persisted via Local Storage

TODO list app created by Harold Sikkema in the context of Systems Design (Databases and Dynamic Data) in the Interaction Design Program at Sheridan College. This app uses Local Storage to persist a collection of TODO items in the browser. Data storage is handled via a set of functions: createData, readData, updateData, deleteData, as per the the acronym CRUD. To keep things modular, functions are imported from a separate module, crud.js
